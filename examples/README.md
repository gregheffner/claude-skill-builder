# Examples

Real skills generated by `skill-builder-agent.py`. Drop into your own
`.claude/skills/` to use, or read them to see what a complete skill looks
like.

## `youtube-transcript/`

Built from the prompt:

> *"fetch youtube transcripts from a URL or search query"*

The agent picked `yt-dlp` for the heavy lifting, locked tools down to
`[Read, Write, Bash, AskUserQuestion]`, and wrote out the VTT-cleaning awk
pipeline itself. Total time from prompt to working skill: ~30 seconds.

To use it in your own setup:

```bash
brew install yt-dlp
mkdir -p .claude/skills
cp -r examples/youtube-transcript .claude/skills/
```

Then open Claude in that directory and paste a YouTube URL.
